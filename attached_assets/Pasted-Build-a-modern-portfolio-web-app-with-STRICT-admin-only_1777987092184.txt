Build a modern portfolio web app with STRICT admin-only authentication (no public signup, no Google login, no OAuth).

TECH STACK:
- Next.js 14 (App Router)
- Tailwind CSS
- Simple backend using Next.js API routes
- Storage: Supabase (for files + database)

AUTH SYSTEM (IMPORTANT):
- DO NOT use Google login or OAuth
- DO NOT allow user registration
- Create a simple login system using:
  - username + password (hardcoded in environment variables)
- Only 1 admin user exists
- If not logged in → cannot access /admin
- Use JWT or session cookie

ENV VARIABLES:
- ADMIN_USERNAME
- ADMIN_PASSWORD
